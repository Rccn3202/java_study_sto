1. 운영체제란?
시스템 소프트웨어
하드웨어 기능을 관리하는 역할
응용프로그램(어플리케이션) 관리하는 역할
하드-응용 중개자
메모리 관리
응용프로그램(어플같은것)?

2.운영체제의 종류
윈도우(Windows) : 컴퓨터 지식 없어도 사용할 수 있다.
맥 OS(Mac OS)
리눅스(Linux)
유닉스(Unix) 등

3.컴퓨터에서 처리하는 방식에 따른 분류
컴퓨터 처리능력에 따른 분류

컴퓨터 분류방식 3가지

정보처리형태 따른 분류
일괄처리 : 한꺼번에 모아서 일정기간 모아서 처리, 1세대
(아르바이트 시급 계산 등)

실시간 처리: ​자료가 발생하자마자 즉시 처리, 2세대, 온라인
(예약시스템, 영화관 예매)

다중프로그래밍: 한번에 여러개 프로그램 실행
(브라우저, 메모장 여러개 켜는 것)


분산처리 : 작업을 나누어서 처리, 4세대
(처리할 프로그램 100개면 4토막내서 나눠서 처리/지금 컴 다 분산처리)

동배간처리(P2P): 내가 서버/클라이언트가 될수있는 방식

임베디드: 응용프로그램을 탑재하여 컴퓨터기능을 수행

다중처리: 여러개 CPU설치하여 실행

클러스터링: 여러 대 컴퓨터를 단일시스템으로 사용하는 기능

시분할처리:tss 타임 시어링 시스템(3세대 메인 시스템70년대쯤)
(cpu에 실행 명령 하면 웹브라우저 10초, 이클립스 15초/
시분할은 cpu에게 할당시간 5초 줌, 브라우조 5초 후 이클립스 5초 후 브라우저 5초->급한일 처리할 수 있음
일괄처리는 웹브라우저 10초 후 이클립스 15초 수행)

CLI vs GUI
CLI 명령어를 문자로 하기
GUI 명령어를 아이콘(그림)으로

윈도우 cli명령어 
목록보기 : dir
해당폴더 이동하기: cd 폴더명
경로 지정: path  (c드라이브 밑에 java폴더 밑에..)
폴더 지우기: rd (remove directory)

리눅스 명령어
목록보기 :ls (list)
파일 만들기 : touch

인터럽트 : 오류로 인해 컴퓨터가 멈추는 경우 강제로 프로그램을 종료하는 것 등(가로채기)
동작이 정상적으로 안되면 강제로 멈춰주는 것
ctrl alt delete -작업관리자

프로세스 : process, 운영체제(윈도우)가 디스크(c드라이브)에 저장된 프로그램을 메인 메모리(RAM)로 로딩(불러오는 것을 시키는)하는 것
프로세서 : processor, cpu를 부르는 말, 프로세스를 실행시키는 기계(하드웨어,cpu)

부팅 : 컴퓨터를 사용할 수 있게끔 정상적으로 켜주는 상태

운영체제 작업 우선순위(스케줄링)
여러 명령이 들어왔을 때,  우선순위 선택기준
1 선입선처리 스케줄링(FCFS) (First Come First Served, First In First Out)
먼저 들어온 것 먼저 수행
4. 라운드 로빈 스케줄링(Round-Robin)/ 시분할과 비슷
할당시간 5초씩
고정된 시간값을 가지고 주어진 시간을 선점하는 프로세스를 실행함
선점: 중간에 처리하다가 덜끝났는데 cpu할당에 나오는 것(라운드로빈)
비선점:cpu처리 들어가면 끝까지 다 하고 나오는 것(선입선출 FCFS)
