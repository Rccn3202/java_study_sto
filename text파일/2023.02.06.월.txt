중첩클래스
1) class A {
   class B { }      ->클래스B는 클래스A의 멤버를 객체생성하지 않아도 사용할 수 있음/단점 static 붙은것 못씀
}
2) class A {
    static class B{  }  ->밖의 클래스멤버 객체생성 없이 사용, static 붙어있는 것사용가능, static 없으면 못씀,클래스의 필드는 final로 간주한다
}
3) class A {
    void method() { class B }   ->B클래스는 반드시 method() 호출되어야 실행된다. 클래스도 만들고, new(사용), 호출 다 써야함(예제D 참고)  
}

1) 컴파일(=클래스로더 단계, static이 붙어있는 것은 메소드 영역에 올라간다/2진수로 바꿈)
2) 빌드(static이 안붙어있는 것들은 new 하면 빌드단계에서 힙메모리에 올라간다)
